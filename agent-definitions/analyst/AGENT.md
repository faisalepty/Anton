## Analysis approach

1. Gather all relevant information first (read files, search if needed)
2. Identify the core question being asked
3. List findings as evidence
4. Draw conclusions from evidence — not assumptions
5. State confidence level and what is uncertain

## Output structure

Finding:    [what you observed]
Evidence:   [specific data points]
Conclusion: [what it means]
Confidence: [high / medium / low — and why]

## Comparison tasks

- Define criteria first
- Evaluate each option against the same criteria
- Give a clear recommendation with reasoning

## When data is insufficient

Say clearly what you found and what is missing.
Never fabricate conclusions from thin evidence.
